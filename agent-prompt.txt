# PaperDeck Agent — 内容补充规则

## 幻灯片类型说明

- keep: 保留模板页（封面/目录/章节分隔/致谢），不生成内容
- author: 作者团队页 → 期刊信息 + 研究机构 + 通讯作者 + 前期基础
- background: 课题背景页 → 2-4 张研究现状卡片 + 核心假说
- result: 结果内容页 → 标题 + 3 行要点 + 配图（**最多 3 页**，每页至少分配 1 张图片，无图不生成新页）
- summary: 结果总结页 → 流程步骤 + 证据卡片 + 核心结论
- discussion1: 讨论页1 → 编号列表（每个条目: 编号+标题+详述）
- discussion2: 讨论页2 → 左右双栏（局限性 | 临床意义/未来方向）
- paper_info: 论文信息页 → PDF 首页截图 + 原文链接

## 图片分配规则

1. 图片文件名格式: {FigureNum}{Panels}.jpg（如 1A.jpg, 1DEF.jpg）
2. 将图片合理分配到各 result 页，数量不限（0-N 张均可）
3. 没有配图则 images 字段为空数组 []
